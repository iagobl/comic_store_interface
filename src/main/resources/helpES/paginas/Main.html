Pantalla Principal

La pantalla principal de la aplicación se compone de las siguientes regiones:
  • Lista de Comics: En el centro de la aplicación se muestra una lista con los comics que hay en la base de datos
  • Lista de Botones: En la parte derecha de la aplicación se muestra una lista de botones donde se puede seleccionar las distintas ventana de la aplicación
    • Boton para ver la lista con los comics de la base de datos
    • Boton para ver la lista con las coleciones de la base de datos
    • Boton para ver la lista con los autores de la base de datos
    • Boton para ver la pestaña con los informes, y seleccionar que tipo de informe quieres
    • Boton para acceder a la ayuda de la aplicación
    • Boton para acceder a la configuración de la aplicación, donde se puede seleccionar el idioma de la misma
  • Apartado de Buscar: En la parte superior de la aplicación apareceria un cuadro de texto que te permitiria buscar por comic, colección y autores
  • Apartado inferior: En la parte inferior de la aplicación aparece dos botones uno de añadir y otro de eliminar, que dependiendo en que pestaña estes te lleva a una pestaña de añadir o otra
